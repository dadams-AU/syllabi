Using This Repo

Course folders keep term-specific LaTeX source files alongside compiled PDFs. Most recent syllabi use the CSUF syllabus template and compile with LuaLaTeX for accessible PDF output.

  1. Browse an existing course folder for a recent term.
  2. Copy the closest syllabus source into a new term folder.
  3. Update course details, instructor information, schedule, policies, and readings.
  4. Compile with LuaLaTeX to preserve PDF tagging.
