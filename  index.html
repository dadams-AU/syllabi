Adams Syllabi Collection

A repository of LaTeX-formatted syllabi for undergraduate and graduate Public Policy and Administration courses that I teach at Cal State Fullerton.

📍 Access the syllabi:

  • 🌐 Web: syllabi.dadams.io
  • 💻 GitHub: github.com/dadams-AU/syllabi

📚 Available Syllabi

🎓 Undergraduate Courses

  • POSC 315: Introduction to Public Policy
  • POSC 320: Introduction to Public Administration

📚 Graduate Courses

  • POSC 509: Foundations of Public Administration
  • POSC 521: Public Administration Theory: MPA Capstone
  • POSC 588: Collaborative Governance

📋 Template

Access the LaTeX template and preview the output to get started with your own syllabus.

📖 Usage Guide

Getting Started

  1. Navigate to the desired course folder
  2. Each folder contains:
    • syllabus.tex - LaTeX source file
    • syllabus.pdf - Compiled document for viewing

🔄 Maintenance

  • Repository contains the most recent versions of all syllabi
  • Regular updates reflect current course content and policies
  • Version history available through commit logs
